# php ajax encuesta

encuesta de usarios usando el metodo AJAX con las siguientes tecnologias:

    1. frontend
        1.1. materialize-css
    2. backend
        2.1 php
        2.2 phpdotenv
        2.3 mysqli

> se conocer de php7, mysql variables de etorno, terminal y el gestor de paquetes composer.
